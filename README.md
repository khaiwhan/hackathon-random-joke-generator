# Random Joke Generator

## รายละเอียด:

สร้างแอปที่มีปุ่ม "Get Joke"
เมื่อกดปุ่ม ให้ดึงข้อมูลจาก API (เช่น Official Joke API) และแสดงผลบนหน้าเว็บ

## เงื่อนไขเพิ่มเติม:

ใช้ Angular Service สำหรับการเรียก API
จัดรูปแบบ Joke ให้ดูสวยงามและอ่านง่าย
